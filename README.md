Bonita Connector
================

Bonita connector allows Bonita BPM to use OpenKM to save and retrieve documents.

More information at:

* http://www.openkm.com
* http://www.bonitasoft.com

OpenKM documentation about connector http://wiki.openkm.com/index.php/Bonitasoft_Workflow_connector
